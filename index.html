About This Dashboard
Campus Crime Dashboard Tool

This interactive dashboard provides visualizations of campus crime data, helping to understand safety trends and patterns. The data is presented in two formats:

  • Daily Statistics: Detailed incident-by-incident data collected from daily/weekly logs with specific dates, times, and locations.
  • Yearly Statistics: Aggregated yearly data collected from annual reports showing long-term trends and patterns across campus locations.

The visualizations include:

Daily Dashboard
  • Incident Type Distribution
  • Reports Per Month
  • Top Crime Locations
  • Day of Week Analysis
  • Time of Day Analysis
  • Incident Status Breakdown
  • Dollar Amount Analysis
Yearly Dashboard
  • Crime Type Distribution
  • Yearly Crime Trends
  • Location Comparison
  • Crime Category Analysis
  • Year-over-Year Analysis
  • Total Offenses by Location
